## 조작법

- 1p : wasd 이동, f - 슛 or 블락, g - 패스 or 스틸.
- 2p : ijkl 이동, ; - 슛 or 블락, ' - 패스 or 스틸.

키 설정은 처음에 게임 실행 시 유니티 런쳐에서 변경 가능합니다.
